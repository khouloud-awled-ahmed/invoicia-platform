# Dépendances requises pour le module d'import bancaire

Pour que le module d'import bancaire fonctionne, vous devez installer les dépendances suivantes :

```bash
cd backend
npm install pdf-parse csv-parse
```

Ces packages sont nécessaires pour :
- `pdf-parse` : Extraire le texte des fichiers PDF
- `csv-parse` : Parser les fichiers CSV

Après installation, redémarrez le serveur backend.
